# Supporting Technical PDFs (Blindaje)

This folder contains complementary technical documents supporting the doctoral thesis and annexes.

## Intended content
- Extended mathematical derivations (e.g., FNRR development)
- Additional tables and diagnostics not included in the main thesis
- Reproducibility reports and logs exported to PDF (optional)
- Conceptual diagrams and technical notes in PDF format

## Policy
- These files are supplementary and meant to strengthen transparency and auditability.
- The core reproducible pipeline is provided in `scripts/` and documented in `docs/anexos/`.
